# Splunk Basics

## Overview

This TryHackMe room introduced Splunk, a Security Information and Event Management (SIEM) platform used by SOC Analysts to collect, search, analyze, and investigate security logs. The room covered Splunk navigation, searching data, creating queries, and analyzing events.

## Skills Demonstrated

- Splunk fundamentals
- Search Processing Language (SPL)
- Log analysis
- Event investigation
- Dashboard navigation
- Security monitoring
- Data filtering and searching

## Key Takeaways

- Learned how to search and filter events using SPL.
- Explored Splunk dashboards and visualizations.
- Investigated log data to identify relevant security events.
- Developed familiarity with a common enterprise SIEM platform.
