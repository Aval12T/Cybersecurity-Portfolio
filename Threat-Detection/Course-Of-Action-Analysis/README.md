# Course of Action Analysis

## Overview

This project analyzes adversary actions throughout the attack lifecycle and identifies defensive countermeasures that can be implemented to reduce organizational risk.

## Objectives

* Analyze attack progression
* Identify adversary tactics and techniques
* Evaluate defensive controls
* Recommend mitigation strategies

## Topics Covered

* Reconnaissance
* Weaponization
* Delivery
* Exploitation
* Installation
* Command and Control
* Actions on Objectives

## Skills Demonstrated

* Threat Analysis
* Cyber Kill Chain
* Defensive Strategy
* Security Operations
* Risk Mitigation
