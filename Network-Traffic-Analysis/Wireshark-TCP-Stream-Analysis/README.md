# Wireshark TCP Stream Analysis

## Objective

Analyze network traffic using Wireshark to identify communications, credentials, and potential security concerns.

## Tools Used

- Wireshark
- Kali Linux

## Activities Performed

- Captured network traffic
- Followed TCP Streams
- Analyzed POP3 traffic
- Reviewed ARP packets
- Examined plaintext communications

## Findings

- Identified email communications transmitted in plaintext
- Observed POP3 authentication traffic
- Reviewed network host communications
- Investigated packet-level activity

## Skills Demonstrated

- Packet Analysis
- Network Traffic Monitoring
- Protocol Analysis
- Security Investigation
