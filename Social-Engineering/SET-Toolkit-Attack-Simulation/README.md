# Social Engineering Toolkit (SET) Attack Simulation

## Overview

This project explores the Social Engineering Toolkit (SET) and demonstrates how attackers can leverage social engineering techniques to obtain credentials and compromise systems.

## Objectives

* Explore social engineering attack methods
* Understand credential harvesting techniques
* Review web-based attack simulations
* Analyze attacker tactics and methodologies

## Tools Used

* Social Engineering Toolkit (SET)
* Kali Linux

## Skills Demonstrated

* Social Engineering Awareness
* Credential Harvesting Concepts
* Attack Simulation
* Threat Analysis
* Security Awareness

## Attack Methods Reviewed

* Credential Harvester
* Web Jacking
* Multi-Attack Web Method
* HTA Attack Method

## Key Takeaways

Understanding social engineering techniques helps security professionals identify, prevent, and respond to attacks that target human behavior rather than technical vulnerabilities.
