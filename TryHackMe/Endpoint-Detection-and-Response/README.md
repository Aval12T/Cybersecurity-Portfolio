# Endpoint Detection and Response (EDR)

## Overview

This TryHackMe room introduced Endpoint Detection and Response (EDR) concepts, including endpoint telemetry, process trees, indicators of compromise (IOCs), threat detection, investigation workflows, and response actions.

## Skills Demonstrated

- EDR fundamentals
- Endpoint telemetry analysis
- Process tree investigation
- Threat detection concepts
- IOC identification
- Endpoint response actions
- Security alert analysis
